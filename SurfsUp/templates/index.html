Vacation in Honolulu App


We are delighted to make your trip to Honolulu, Hawaii as happy as it can ever get!

Available Routes


  • /api/v1.0/precipitation to view Precipitation data.
  • /api/v1.0/stations view the list of Stations.
  • /api/v1.0/tobs make a note of Temperature Observations.
  • /api/v1.0/<start> Write Start Date in the url where there is <start> in the url on the page that opens after clicking this link.
    date range from "2016-08-23" to "2017-08-23"
  • /api/v1.0/<start>/<end> Write Start and End Date where there is <start>, <end> in the url on the page that opens after clicking this link.

About


This is a simple climate app built with Flask. It uses jsonify function to convert API data to a valid JSON response object
